The form tags, and how to use them to get user input and respond to that with jQuery.

Programs are boring if they just do the same thing every time. We need to respond to things the user does to make things interesting. So far, we have one way to do this: respond to user actions by letting jQuery connect the actions to our functions.

That's good but limited: the user clicks on something or they don't. They move the cursor somewhere or they don't. We would like to get richer input from them.

HTML Forms

We can have the user give us more detailed input using the form tags in HTML, starting with <input>.

In previous versions of HTML, all inputs had to be inside a <form> element. That is not true in HTML5 the way we're using inputs.

jQuery and Forms

Once we have some input from the user with a form, we can access it with JavaScript and jQuery.

The easiest thing to do is find the <input> with jQuery (probably selecting by id) and using the jQuery object's .val() method that will give you what the user typed.

In the example above, we had an input with id="something" so we can get the string the use typed with jQuery('#something').val().
